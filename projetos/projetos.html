Explicação do Projeto de Rede

Sala de Reunião: A rede foi configurada para facilitar o uso de notebooks e sistemas de videoconferência. Isso permite uma comunicação eficiente, interna e externa, para reuniões remotas e apresentações.

Salas de Escritório: Cada sala possui pontos de rede conectados via switches, garantindo uma conexão estável para os computadores dos funcionários. Isso é essencial para o desempenho das tarefas diárias.

Sala de Servidores: Centralizada para gerenciar o tráfego de dados da empresa e garantir a segurança. É responsável por armazenar dados e controlar os recursos da rede.

Hall de Entrada: O ponto de rede na recepção está configurado para registrar visitantes, controlar o acesso e se comunicar diretamente com o sistema administrativo.

A disposição da rede foi feita de forma a garantir segurança, eficiência e um fluxo de dados organizado, permitindo que cada setor da empresa tenha uma infraestrutura de TI confiável.
